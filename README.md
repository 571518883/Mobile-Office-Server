## [Mobile-Office 安卓源码](https://github.com/wangcantian/Mobile-Office)

**个人与队友(老王，阿彬)第一个参赛的作品，移动微办公应用，模仿阿里钉钉部分功能，主要有工作签到、公告、审批、会议室4大功能。**

**宗旨：减少了办公人员的工作流程，将工作效率大大的提高，也实现的办公的无纸化**

### 本作品用于个人学习、参赛

### 选取框架：Thinkphp
